Agent mode — tool loop

Subcommands: check · ci · agent · cache-clear. The agent exposes 14 tools across six groups; LLM drives until it returns a final reply or hits MAX_STEPS = 12. aicritic cache-clear wipes .aicritic_cache/.

flowchart TB T([Natural language task]) --> LLM[Claude · tool use] LLM --> TOOLS{Tools · 14 total} TOOLS --> R1["Load: read_files · get_changed_files"] TOOLS --> R2["Analyse: analyse · cross_check · critique"] TOOLS --> R3["Shortcut: run_analysis"] TOOLS --> R4["Fix & ship: apply_fixes · open_pr"] TOOLS --> R5["File ops: read_file · write_file · run_shell"] TOOLS --> R6["Extras: generate_tests · refactor · save_baseline"] R1 & R2 & R3 & R4 & R5 & R6 --> LLM LLM --> DONE([Final reply])

--generate-tests FILE writes runnable tests for high-risk uncovered paths and tracks coverage trend in .aicritic-coverage-history.json. Tests carry a “# generated by aicritic” guard and are never committed unless --auto-commit-tests is set.

Copilot Extension — POST /

FastAPI app at server.py. Per-request user token is used for model calls so org Copilot licence covers usage.
